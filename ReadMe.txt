The Online Banking System Code contains the folllowing text files:
1)accountno.txt:Contains random account numbers.An account no will be given to any one who opens an account.
2)SavinAccountDetails.txt: Contains the Name, Email, Contact no., Password,Account balance and Account number of the cutomers.
3)CheckingAccountDetails.txt: Contains the Name, Email, Contact no., Password,Account balance and Account number of the cutomers.
4)LoanAccountDetails.txt:This File stores the detail of a loan account.The Account Number,Loan_amount and the repayment period is saved in the file as a list.  
5)TransactionHistory.txt: This file has the detail of the transaction made by an accountnumber with the type of transaction(either with draw or deposit)and date and time. 
6)Issues.txt: This file stores the issues of the customers in a list. The first index of the lists have the complainant and the issues are written in the second index.
